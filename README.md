# WiRays® Coverage — iOS App

iOS app (Capacitor 6 + WKWebView) wrapping the WiRays® Coverage visualization. Provides the full 3D + 2D coverage tool as a native iOS app, with offline support, home screen icon, and Mac Catalyst support.

## Stack

- [Capacitor 6](https://capacitorjs.com/) — iOS native wrapper
- Three.js r128 (classic script tags, CDN) inside WKWebView
- `@capacitor/filesystem` + `@capacitor/share` for file export
- Xcode 15+ required for iOS 17+ target

## Structure

```
www/index.html          Single-page app — all JS inline, no build step
www/favicon.png         Lumeova app icon (also used as apple-touch-icon)
capacitor.config.json   App ID: com.lumeova.coverage
package.json            Capacitor CLI + plugin dependencies
ios/                    Xcode project (generated by Capacitor)
ios/App/App/Assets.xcassets/AppIcon.appiconset/
                        AppIcon-512@2x.png — 1024×1024 PNG, no alpha channel
```

## Development Setup

### Prerequisites

- Node.js 18+
- Xcode 15+ with iOS 17 SDK
- CocoaPods (`sudo gem install cocoapods`)

### Install dependencies

```bash
cd wirays-coverage-app
npm install
```

### Edit web assets

All app logic is in `www/index.html`. Edit directly — no build step required.

### Sync to iOS after web changes

```bash
npm run sync
# or: npx cap sync ios
```

### Open in Xcode

```bash
npm run open
# or: npx cap open ios
```

Then select a simulator or device and hit **⌘R** to build and run.

## Deployment (iOS)

1. Edit `www/index.html` as needed
2. `npx cap sync ios` — copies web assets into the Xcode project
3. Open Xcode (`npx cap open ios`), set your team in Signing & Capabilities
4. Archive → Distribute App for App Store or TestFlight

## App Icon

The app icon is `ios/App/App/Assets.xcassets/AppIcon.appiconset/AppIcon-512@2x.png`.

Requirements:
- **1024×1024 px**
- **No alpha channel** (iOS rejects icons with transparency)

To regenerate from a source PNG with alpha:
```python
from PIL import Image
img = Image.open('source.png').convert('RGBA')
bg = Image.new('RGBA', img.size, (255, 255, 255, 255))
bg.paste(img, mask=img.split()[3])
bg.convert('RGB').save('AppIcon-512@2x.png')
```

## Features

### View Modes (top-bar pill)
- **3D** — full-screen Three.js 3D viewport
- **3D|2D** — split view: 3D left, 2D floor plan right (landscape) / stacked (portrait)
- **2D** — full-screen 2D floor plan

### Settings Drawer
- Floating hamburger button (bottom-right) opens a slide-in panel from the left
- Room dimensions (X width, Y depth, Z height) — no upper cap
- Beam group parameters (center / middle / outer): count, tilt angle, diffuser angle, power per emitter (mW)
- Total group power label updates live: `N × P mW = total mW`
- Plane of Interest (Z') slider
- Scene intensity sliders (beam, grid, Z' plane)

### Zoom (Mac / iPad)
- **⌘+** / **⌘-** — zoom in/out in 10% steps
- **⌘0** — reset to 100%
- Zoom range: 50%–300%
- Zoom persists across sessions via `localStorage`
- Mac-only: zoom activates on first `metaKey` event (never fires on iPhone)

### Watermark
- `© 2026 Lumeova` diagonal tiled watermark at 5% opacity on both 2D and 3D views
- Composited into PNG exports

### Click-to-Measure
- Tap anywhere on the 2D canvas to place a measurement point
- Shows H (horizontal), V (vertical), S (slant), θ (elevation angle)
- Right-triangle diagram in measure box; triangle overlay in 3D view

### Export
- 3D PNG, 2D PNG, Settings JSON — via share sheet (`@capacitor/share`)
- Zemax NSC (.zmx) — OpticStudio Non-Sequential source + detector file
- Cross-browser save-as modal for desktop/Mac

## Development Notes

- All app logic lives in `www/index.html` — one large IIFE, no framework, no build step
- CSS uses `rem` units throughout; root `font-size` scaled by `applyZoom()` for ⌘+/- support
- `touch-action: none` on `input[type=range]` prevents iOS from intercepting slider drags as scrolls
- `renderer.preserveDrawingBuffer = true` required for 3D PNG export
- Room dimension inputs accept 0 as minimum, no upper cap — JS fallback handles NaN/empty only
- `drawWatermark(ctx, W, H)` renders tiled copyright text; `get3DWithWatermark()` composites for export
- `updatePowerLabels()` keeps `N × P mW` labels live as count/power change
- `buildZemaxExport()` generates Zemax NSC `.zmx`; Euler angles derived from beam direction vector
- `touch-action: none` on sliders prevents iOS scroll interference during drag
